Typography

Proportional text

The Maven Pro font family is the only available proportional font family and is set for all proportional text. No other font families should be used for proportional text.

As the Maven Pro family has no italic styles, italics should not be used. Ever.

The primary Maven Pro font styles are included using the Google Fonts API.

Lighter weights of Maven Pro are only available in OpenType format from Lost Type Co- op and not Google Fonts, so have been converted to TrueType (TTF) and Web Open Font Format (WOFF), then included in assets.

Monospaced text

Monospaced text is set on a subset of standard monospaced elements. Standard system fonts (rather than webfonts) are used.

Monospaced text should always use the default weight and size.

Typographic enhancements

typogr.js "provides a set of functions which automatically apply various transformations to plain text in order to yield typographically- improved HTML".

All HTML should be run through typogr at build.

The following Typogr options are enabled:

widont

Replaces the space between the last two words in a string with &nbsp; to avoid a final line of text with only one word.

smartypants

Fixes quotes ( " and ' '"), dashes (“--” and “---”) and ellipses (“...”).

caps

Wraps MULTIPLE CAPITAL LETTERS in <span class="caps"></span> so they can be styled.

Block elements

Typographic elements do not come with any margins attached to them in order to make them more flexible in where they can be used.
